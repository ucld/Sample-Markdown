## ./extra

The ./extra folder contains:
* [demo.html](./demo.html) - Small interactive demo.
* [Parser.Bloat.js](Parser.Bloat.js) - Added Image and Code support.
* [Parser.React.js](Parser.React.js) - Modifications for React Framework.
